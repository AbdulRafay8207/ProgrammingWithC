To compile C in Linux, first run this command:
1. sudo apt install build-essential
then
2. gcc (filename) -o (filename without extension)
by running command 2, you will get a file with no extension.
3. ./(filename with no extension) and you will see your result

Auto:
1. make .zed folder in root project folder and inside .zed add tasks.json which should contain this:
[
  {
    "label": "Run C (Linux)",
    "command": "gcc $ZED_FILE -o ${ZED_STEM} && ./${ZED_STEM}",
    "use_new_terminal": false,
  },
  {
    "label": "Build Windows EXE",
    "command": "x86_64-w64-mingw32-gcc $ZED_FILE -o ${ZED_STEM}.exe",
    "use_new_terminal": false,
  },
]
2. Press ctrl + shift + p, go to kepmap editor then keymap.json and paste this:
[
  {
    "context": "Editor",
    "bindings": {
      "ctrl-'": ["task::Spawn", { "task_name": "Run C (Linux)" }],
    },
  },
]
Now when you will press ctrl + ' , it will auto compile your code.
